使用hexo和github page 搭建免费blog系统

Wenjin
2023-01-20
运维

我的blog一直使用wordpress。服务器一直也蹭朋友的服务器。朋友的服务器终于要停服了。 我也需要为我很少更新blog找一个永不停服的服务器了。
最终选定方案为白嫖github page。 使用hexo创建静态的blog页面，然后通过github page免费发布，通过cloudflare CDN进行加加速。
